# The Digital Detox - Educational Puzzle Game

## Concept
An interactive mobile game that challenges users with real-world research tasks, puzzles, and educational quests across multiple categories like internet research, science, history, and more.

## Tech Stack
- **Frontend**: Flutter (Dart)
- **Backend**: Dart (Shelf framework)
- **Database**: Turso (SQLite)
- **Hosting**: Render (Free tier)

## Game Features
- Multiple categories (Internet Detective, Science Explorer, History Hunter, etc.)
- Progressive difficulty levels (Beginner → Expert)
- Daily challenges
- Hints system with coins/points
- Leaderboard & achievements
- Streak tracking
- Beautiful, accessible UI

## Project Structure

the-digital-detox/

## Game Categories
1. Internet Detective - Web research & fact-finding
2. Science Explorer - Human body, biology, physics
3. History Hunter - Historical events & figures
4. Tech Puzzles - Decode, encrypt, logic
5. Nature Quest - Geography, animals, environment
6. Culture Voyage - Languages, traditions, arts
7. Math Mysteries - Number puzzles, patterns

## Level System
- **Beginner (1-10)**: Simple searches
- **Intermediate (11-30)**: Multi-step research
- **Advanced (31-50)**: Complex puzzles
- **Expert (51+)**: Time-bound challenges
